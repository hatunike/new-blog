### iKennd.ac Blog

The site is generated using `nanoc`.

To install:

1. `$ gem install nanoc`
2. `$ gem install adsf`
3. `$ gem install kramdown`
4. `$ gem install stringex`
5. `$ gem install builder`
6. `$ gem install pygments.rb`

To generate:

1. `$ cd <dir>`
2. `$ nanoc`
3. `$ nanoc view` to view locally.
